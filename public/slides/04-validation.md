# Validation

Answer **3 questions** before writing a single line of code:

---

**1. Is this buildable in time?**
Core feature only — 12–18 hours max

**2. Is this demonstrable to a judge?**
3-minute, clear user flow

**3. Does it fit the scoring criteria?**
Map to the rubric before building

---

> **Pivot Rule:** If you're 30% into build time and the core isn't working — narrow scope, don't restart.
